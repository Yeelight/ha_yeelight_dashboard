## Layout Rules

- Legacy Grid uses `gridX`, `gridY`, `gridW`, `gridH`, 12 columns, collision handling, sparse movement, and compaction.
- Legacy Free uses percentage `x`, `y`, `w`, `h`, and `z`, with overlap and absolute placement.
- New default migration maps Grid to HA Sections and `grid_options`.
- New advanced migration maps Free to Yeelight Canvas View `view_layout`.
